## Vip视频解析

> 通过收集网络中的解析接口来解析视频，本身不存储视频，视频收集自网络

> 点击视频页面弹出的vip按钮然后选择视频解析即可

![AhExNW7HMy3tfdV](https://i.loli.net/2020/04/22/AhExNW7HMy3tfdV.png)

## 音乐下载
